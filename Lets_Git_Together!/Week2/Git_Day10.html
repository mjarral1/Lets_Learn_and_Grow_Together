Task 1: Create the Wiki Repository:

1.Bano sets up a new GitHub repository named TeamGitWiki:
->Navigate to GitHub and click New Repository.
->Name it TeamGitWiki
->Add a README.md file and initialize the repository.

2.Clone the repository locally:
git clone https://github.com/your-username/TeamGitWiki.git
cd TeamGitWiki

Task 2: Add Wiki Pages

1. Jane creates the first Markdown file:
->Create a file named basic-commands.md
->Add information about basic Git commands:
# Basic Git Commands
- `git init`: Initialize a new repository.
- `git status`: Show the status of changes.
- `git add`: Stage files for a commit.

2.Raja adds a section on branching:

-> Create a file named branching.md with content:
# Branching in Git
- `git branch`: List all branches.
- `git checkout -b branch_name`: Create and switch to a new branch.
- `git merge`: Merge branches together.

Task 3: Collaborate Using Branches

1.Each team member creates their own branch:
git checkout -b mary-wiki
2. Add and commit changes:
git add .
git commit -m "Added basic Git commands page"
3. Push changes to GitHub:
git push origin mary-wiki

Task 4: Submit and Merge Pull Requests:

1.Mary opens a pull request on GitHub:
->Navigate to the mary-wiki branch.
-> Click Pull Request > Compare & Pull Request.
-> Add a description and assign team members to review.
2. The team reviews and merges the pull request into the main branch.
